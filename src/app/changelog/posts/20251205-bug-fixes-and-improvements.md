This week's release brings several UX improvements, and a set of stability fixes across the app.

## More improvements

- Enhanced Simulation Form on Mobile for improved readability and overall usability.
- Added inner data types in the decoded data view to make complex structures easier to understand and debug.
- Fixed a background rendering issue in Call Trace Search, improving visual display.
- Corrected date and time alignment for Contracts on the Class page.
- Fixed an issue that blocked transaction simulation when relocated trace data was missing.
- Updated [Walnut Verification documentation](https://docs.walnut.dev/overview/verify-starknet-contracts-in-walnut) to reflect the latest `sncast verify` workflow.
